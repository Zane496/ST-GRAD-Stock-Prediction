# ST-GRAD:  Multi-View Cross-Attention and Bidirectional Diffusion for Stock Trend Prediction

## 📖 Introduction

Stock market prediction is a complex and challenging task due to the highly noisy and non-stationary nature of financial data. ST-GRAD (Spatio-Temporal Graph Adaptive Diffusion) is a novel deep learning framework designed to overcome the limitations of single-perspective modeling.

The model introduces two core structural innovations:
1. **Multi-View Cross-Attention (MVCA):** Explicitly disentangles multi-dimensional market dependencies into three orthogonal views: spatio-temporal, feature-spatial, and feature-temporal.
2. **Bidirectional Diffusion Graph Convolution (BDGC):** Fuses a static prior graph (derived from deep behavioral clustering) with a dynamically learned adaptive graph to simultaneously capture stable long-term sector structures and short-term market dynamics.

## 🚀 Quick Start & Reproducibility

For the convenience of reviewers and researchers, we provide a unified pipeline script (`main.py`) that handles data preprocessing, graph construction, training, and evaluation.

You can run the entire pipeline end-to-end with a single command:

```python
python main.py --mode all
```

Alternatively, you can execute the pipeline step-by-step to inspect intermediate outputs:

### Step 1: Graph Construction

Builds and symmetrically normalizes the adjacency matrix based on the trend clustering results. The output is saved as `data/processed/normalized_adj_matrix.npy`.

```python
python main.py --mode graph
```

### Step 2: Data Preprocessing

Generates dynamic trading labels based on market volatility and extracts 24 technical indicators (including MACD, RSI, ATR, Bollinger Bands). The output is saved as `data/processed/stock_dataset.npz`.

```python
python main.py --mode data --data_dir data/raw
```

### Step 3: Model Training

Initializes the ST-GRAD model and trains it using the preprocessed dataset and graph matrix. The script applies early stopping based on the Validation F1 Score. The best model weights will be saved in the `models/saved/` directory.

```python
python main.py --mode train --epochs 50 --batch_size 32
```

### Step 4: Model Evaluation (Inference)
To evaluate a pre-trained model on the validation/test set and obtain metrics (Accuracy, Precision, Recall, F1 Score, AUC) without retraining, use the `predict` mode. 

We provide our best pre-trained model weights (`best_model_ours.pth`) for direct verification:
```python
python main.py --mode predict --load_path models/saved/best_model_ours.pth
```

**Note:** `best_model_ours.pth` is the best-performing model weight file reported in our paper (manually renamed for clarity), while `st_grad_best_model.pth` is the default filename automatically generated by the training script.

## 📊 Core Hyperparameters

The performance of ST-GRAD can be tuned via several key hyperparameters in `main.py`:

- `--hidden_channels`: Number of hidden dimensions (default: 32).
- `--num_layers`: Number of stacked Spatio-Temporal Blocks (default: 2).
- `--diffusion_steps`: Maximum steps for the Diffusion Convolution Layer (default: 2).
- `--embed_dim`: Dimension for the Adaptive Adjacency Matrix generator (default: 10).
